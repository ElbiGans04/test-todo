# How To Start ?
### Instal Package First
#### 1. Run Command from command line :
- Yarn
  - `yarn install`
- Npm
  - `npm install`

#### 2. Run Command from command line :
- Yarn
  - `yarn prepare`
- Npm
  - `npm run prepare`

#### 3. Run Command from command line :
- To start development
  - npm 
    - `npm run dev`
  - yarn 
    - `yarn run dev`
- To build
  - npm 
    - `npm run build`
  - yarn 
    - `yarn run build`
- To run the build
  - npm 
    - `npm run start`
  - yarn 
    - `yarn run start`
